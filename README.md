# No Fluxo UnB - Grupo Betty Snyder

Projeto desenvolvido pelo grupo Betty Snyder para a disciplina de Qualidade de Software da Faculdade de Ciências e Tecnologias em Engenharia da Universidade de Brasília (FCTE-UnB), durante o semestre 2026.1.

O No Fluxo UnB é uma aplicação focada em sustentabilidade e eficiência, utilizando características de qualidade de software para otimizar fluxos, organização e experiência do usuário.

---

## Documentação

A documentação do projeto está disponível em: https://fcte-qualidade-de-software-1.github.io/2026-1_T01_BETTY_SNYDER/

---

## Tecnologias utilizadas

- MkDocs
- Material for MkDocs
- GitHub Pages

---

## Como executar localmente

### Instalar dependências

```bash
pip install mkdocs mkdocs-material
```

### Executar o projeto

```bash
mkdocs serve
```

A aplicação ficará disponível em:

```txt
http://127.0.0.1:8000
```

---

## Publicação

O deploy da documentação é realizado utilizando:

```bash
mkdocs gh-deploy
```
