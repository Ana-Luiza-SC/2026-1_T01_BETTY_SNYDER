# No Fluxo UnB - Grupo Betty Snyder

## Sobre o projeto

Esta página é dedicada à documentação de todos os os artefatos produzidos pelo grupo Betty Snyder durante 2026.1, na disciplina de Qualidade de Software da Faculdade de Ciências e Tecnologias em Engenharia da Universidade de Brasília (FCTE-UnB).

O projeto desenvolvido pelo grupo será o [No Fluxo UNB](https://no-fluxo.crianex.com/), uma aplicação utilizando como base duas características de qualidade de software: a **eficiência** e a **usabilidade**.

## Sobre Betty Snyder Holberton

Betty Snyder Holberton foi uma das programadoras pioneiras do ENIAC, destacando-se por sua capacidade de transformar problemas matemáticos complexos em instruções lógicas para a máquina em uma época sem linguagens de programação ou manuais.

Ela introduziu ideias inovadoras como rotinas reutilizáveis e pontos de interrupção (*breakpoints*), que se tornaram práticas fundamentais no desenvolvimento de software.

Além disso, contribuiu para o UNIVAC I, o primeiro computador comercial, e participou de discussões que influenciaram a criação do COBOL, ajudando a consolidar linguagens de mais alto nível.

Suas contribuições foram essenciais para tornar a programação mais acessível, eficiente e estruturada, marcando profundamente a evolução da computação.
